# Frontend CSS

## When to use this skill

- When styling UI components or creating layouts
- When working with CSS frameworks (Tailwind, Bootstrap, etc.)
- When defining design tokens (colors, spacing, typography)
- When applying CSS methodologies (BEM, utility classes, CSS Modules)
- When writing custom CSS or preprocessor code (Sass, Less)
- When using CSS-in-JS solutions (styled-components, Emotion)
- When optimizing CSS for production (purging unused styles)
- When maintaining a design system or style guide
- When working with CSS files (.css, .scss, .sass, .less)
- When applying Tailwind utility classes in components
- When configuring CSS framework settings or themes
- When minimizing framework style overrides

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend CSS.
