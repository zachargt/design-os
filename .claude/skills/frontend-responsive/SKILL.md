# Frontend Responsive

## When to use this skill

- When creating layouts that need to adapt to different screen sizes
- When implementing CSS media queries and breakpoints
- When designing mobile, tablet, and desktop views
- When using fluid layouts with percentage-based widths
- When choosing relative units (rem, em) over fixed pixels
- When ensuring touch targets are appropriately sized (44x44px minimum)
- When optimizing images and assets for mobile networks
- When maintaining readable typography across all breakpoints
- When prioritizing content for smaller screens
- When testing UI changes across multiple device sizes
- When working with responsive CSS frameworks (Tailwind breakpoints, Bootstrap grid)
- When implementing mobile-first development patterns

This Skill provides Claude Code with specific guidance on how to adhere to coding standards as they relate to how it should handle frontend responsive.
